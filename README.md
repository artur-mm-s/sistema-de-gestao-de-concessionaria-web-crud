# SISTEMA DE GESTÃO DE CONCESSIONÁRIA WEB (CRUD) — 2025

O repositório contém o código-fonte de um projeto prático realizado durante o meu **segundo semestre** no curso de Engenharia de Software.

O código se refere a um trabalho cujo objetivo é simular um **Sistema de Gestão de Concessionária** completo, implementando todas as operações de **CRUD** (Criação, Leitura, Atualização e Exclusão) para as principais entidades do negócio. Foi uma oportunidade de evoluir conhecimentos em desenvolvimento Web e integração com banco de dados.

## Tecnologias Utilizadas

* **Linguagem Principal:** PHP (Lógica de Backend)
* **Banco de Dados:** MySQL (Utilizando a extensão `mysqli` para conexão e comandos SQL)
* **Frontend:** HTML e CSS (Com a utilização de **Bootstrap** para um layout responsivo e moderno)

## Funcionalidades Principais

O sistema gerencia as seguintes entidades, todas com módulos de Cadastro, Listagem, Edição e Exclusão:

* **Clientes:** Cadastro completo de informações de compradores, incluindo nome, CPF, e contato.
* **Funcionários:** Gestão dos dados da equipe de vendas (vendedores).
* **Estoque de Veículos (Marcas e Modelos):**
    * Cadastro de **Marcas**.
    * Cadastro de **Modelos/Veículos**, registrando atributos como nome, cor e ano (o sistema foi construído **sem o módulo de fotos**).
* **Gestão de Vendas:** Módulo de transações que relaciona o Cliente, o Funcionário (Vendedor) e o Modelo de Veículo vendido, com registro de data e valor.
